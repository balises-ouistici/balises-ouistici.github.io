Balise Pico-CC1101 simple

Matériel nécessaire

  • module CC1101
  • Raspberry Pico
  • Haut-Parleur 4ohms
  • Module Amplificateur MAX98357A
  • Boîtier étanche (bloc de dérivation 15cmx15cm par exemple)
  • Bouton lumineux 5V

Programme

Les sources sont ici : https://github.com/balises-ouistici/circuitpython-nfs32002

  2. Préparer le Pico pour le programmer en CircuitPython (voir la recette ici)
  3. Charger le programme dans le Pico
