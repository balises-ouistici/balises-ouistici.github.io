# Balise Pico-CC1101 bouton

## Matériel nécessaire

- module CC1101
- Raspberry Pico
- Haut-Parleur 4ohms
- Module Amplificateur MAX98357A
- Boîtier étanche (bloc de dérivation 15cmx15cm par exemple)
- Bouton lumineux 5V
